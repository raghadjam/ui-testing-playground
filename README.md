
# UI Testing Playground

This repository contains Cypress tests for practicing UI interactions, handling dynamic content, and verifying element behaviors across multiple web pages.

## Covered Pages and Tests

* **Load Delay** – Waits for a page element to load before clicking. 
* **Client Side Delay** – Confirm messages appear after data loading.
* **Click** – Verify buttons ignoring DOM click events.
* **Text Input** – Update button text from input and handle empty input.
* **Scrollbars** – Scroll elements into view before interaction.
* **Dynamic Table** – Read and compare dynamic table values.
* **Verify Text** – Match elements while normalizing whitespace.
* **Progress Bar** – Wait for progress bar to reach a target value.
* **AJAX Data** – Confirm messages appear after AJAX loading.
* **Welcome Text** – Verify nested text content correctly.
* **Hidden Layer** – Handle elements not visible initially.
* **Dynamic ID:** Finds elements with changing IDs using stable selectors.  

## Setup

```bash
npm install
npx cypress open
```
